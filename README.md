# House-Price-Prediction-ML
End-to-end house price prediction using regression

## House Price Prediction Using Machine Learning

### Problem Overview
This project predicts house prices based on property and location features using regression techniques.

### Dataset
The dataset contains housing features such as area, bedrooms, bathrooms, parking, and amenities.
Data was collected from a dataset at kaggle

### Machine Learning Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
